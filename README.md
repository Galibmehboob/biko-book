# Biko Book

## Live URL

https://biko-book.vercel.app/

## Purpose

Biko Book is an online book borrowing platform designed to digitize the traditional library experience. Users can browse, search, and borrow books easily through a modern and responsive interface.

## Key Features

- User authentication (Email/Password and Google login)
- Browse all books with category filtering
- Search books by title
- View detailed book information (private route)
- Borrow books with availability tracking
- User profile with update functionality
- Fully responsive design
